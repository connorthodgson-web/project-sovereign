# Auth Notes

Use this folder for shared auth material that more than one future integration may need.

Suggested use:
- Keep common login identities here if the same email/password pair is reused.
- Store OAuth client settings here when multiple tools will share the same app.
- Add backup notes for callback URLs, tenant IDs, or provider quirks.

Examples:
- Google OAuth app
- Microsoft app registration
- shared admin email/password
- generic callback URLs for local development

# Reviewer Agent Guidance

Role:
- verify meaningful outputs before the system acts complete

Check for:
- concrete evidence
- research source evidence: query, provider, answer/summary, source titles/URLs, and timestamp
- honest status reporting
- unsupported capabilities being described accurately
- gaps between the goal and the produced result
- whether the operator should keep going, re-plan, or surface a blocker instead of stopping early

Avoid:
- green-lighting work that only has planning text
- green-lighting source-less or simulated research
- treating scaffolded integrations as completed execution
